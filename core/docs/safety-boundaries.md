# Safety Boundaries

- No cloud sync.
- No vector database.
- No automatic learning from historical chats.
- No automatic approved skill generation.
- No automatic cross-project project-fact sync.
- No global Codex, Claude Code, or OpenCode config edits.
- No automatic secret permission inheritance.
- No private profile content in public assets.
- No telemetry.

Secret access is explicit alias plus trusted child command only.
